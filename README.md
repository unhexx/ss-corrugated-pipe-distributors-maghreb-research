## Методология
- Несколько циклов по 5 параллельных агентских сессий (subagent_type=general-purpose + researcher persona).
- Инструменты: web_search (многоязычные запросы), open_page / browse (обязательная верификация сайтов с цитатами), параллельный сбор.
- Ключевые слова: distributeur / grossiste / importateur + "tuyau flexible inox" / "flexible ondulé" / "âme ondulée inox" / "corrugated stainless hose", Swagelok/Nortec партнёры, O&G suppliers, арабские эквиваленты, городские и отраслевые запросы.
- Строгий фильтр: только компании с ролью дистрибьютора + доказательствами продукта + локальным присутствием.
- Верификация: прямой просмотр страниц продуктов, about, каталогов.

## Рекомендации по использованию
1. Приоритет — позиции с **Высокий** и официальные партнёры премиум-брендов (Nortec, Swagelok).
2. Для каждого кандидата: подготовить предложение с техническими характеристиками (диаметры, давление, температура, сертификаты).
3. Ливия и Египет: учитывать проектный характер (O&G, тендеры, NOC-подобные структуры).
4. Дополнительно использовать местные торговые палаты, отраслевые выставки (NAPEC и др.).
5. Прямая верификация наличия и условий обязательна.
